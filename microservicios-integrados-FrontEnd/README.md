# Proyecto Monolino-plataformaonline

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.2.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Development server render

Navigate to `https://monolito-plataformaonline.onrender.com/inicio`. The application will automatically reload in render.

## Build

Run `ng build` local en render `ng run build` to build the project. The build artifacts will be stored in the `dist/` directory.
